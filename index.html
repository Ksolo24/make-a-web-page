Basics of the Web

The Web

The www(World Wide Web) is a collection of HTML documents, with over 30 billion pages in total. HTML documents are made up of something called HTML(Hypertext Mark up Language), a language that web browsers can read & compose them into visible or audible web pages. HTML documents are made up of tags and elements, browsers do not display the HTML tags and scripts, but use them to interpret the content of the page.

How a Web Page Works

When a user wants to access a web page he runs a browser(for example, Google Chrome) to display HTML documents found on the web, through the internet which connects it to the server.The internet is the world's largest computer network & a server is a computer that host the HTML files that make up the web page. The internet does this by following a set of rules called HTTP, a protocol of the web.

Creating a Structured Document

HTML & CSS


- HTML and CSS are both languages.
- HTML controls the "structure" of a webpage.
- CSS controls the "style" of a page (how it looks).
- When programmers talk about the "DOM"(Document Object Model) they are talking about the "tree-like" structure of a page.

HTML


HTML language = syntax + rules.
- Written in a document just like regular text but instead of text, you follow HTML syntax and rules.
- The browser turns HTML tags into elements that form a tre, it knows how to dot hat beacuase of the DOM.

DOM


- Standard convention for representing and interactive with the elements in the HTML.
- Each HTML tag creates a element in the DOM that the browser uses to display the page.

CSS


- Styling the page.
- The HTML document can provide the browser with information on what style to use and where to get it.

In creating a structured document you can use Scratchpad, Codepen or Sublime. These are some applications you can use upon creating a HTML/CSS document.

Adding CSS Style

How to include CSS styling in a webpage.


1.) Write CSS in the Head element of your HTML.
2.) Link your HTML to a separate CSS file.
-This is done by writing your HTML and CSS in a separate file. Add a Link tag inside the Head element.
3.) Write your style inline with your HTML.

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements.
